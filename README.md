# Chao

Chao greets you with an English phrase "hello".

## Installation

Add this line to your application's Gemfile:

    gem 'chao'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chao

## Usage

Just run

Chao.hello

This will return a string "hello."
